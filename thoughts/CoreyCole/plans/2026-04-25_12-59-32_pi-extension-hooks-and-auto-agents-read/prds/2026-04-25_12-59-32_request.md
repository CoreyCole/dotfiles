# Request

Plan new Pi extensions for the personal dotfiles Pi config.

Requested capabilities:

1. Pre/post tool use hook config
2. Auto documentation reading extension that automatically reads `AGENTS.md` in directories where files are read
3. If a file is read, all relevant `AGENTS.md` files should also be read
4. Each unique `AGENTS.md` should only be read once per session

Initial working context:
- Repository: `dotfiles`
- Pi config lives under `.pi-config/`
- Relevant likely areas include extension hooks, tool call lifecycle events, and resource/context loading behavior
