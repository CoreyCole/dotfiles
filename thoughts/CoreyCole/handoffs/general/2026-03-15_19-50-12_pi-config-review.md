## Learnings
- **AGENTS.md is pi-specific**: Pi reads `AGENTS.md` from `~/.pi/agent/` as its system prompt. It is NOT auto-read by Claude Code (which only auto-reads `CLAUDE.md`). For project-level configs, `CLAUDE.md` is the cross-tool standard (Claude, Codex, Gemini all read it).
- **Pi is not yet installed**: The `pi` CLI is not on this machine. `setup.sh` and package installation are deferred until pi is installed.
- **Symlink pattern**: `.pi-config/` lives in dotfiles repo, symlinked to `~/.pi/agent/` ‚Äî mirrors the `.agents/` ‚Üí `.claude/{agents,skills}` pattern.
